Google Gemini API

Harnessing the power of Gemini for advanced data engineering tasks.

1. Text Summarization with Google Gemini API

In modern data engineering, summarizing large volumes of unstructured text is a common requirement. Using the Google Gemini API, we can build scalable pipelines to process documents and extract key insights efficiently.

Python 
import google.generativeai as genai
import os

# Configure the API key from environment variables
genai.configure(api_key=os.environ["GEMINI_API_KEY"])

def summarize_technical_docs(text):
    # Initialize the Gemini Pro model
    model = genai.GenerativeModel('gemini-pro')
    
    prompt = f"Summarize the following technical documentation concisely:\n\n{text}"
    
    # Generate content
    response = model.generate_content(prompt)
    return response.text

# Example usage for a data pipeline
raw_text = "Generative AI is transforming how we handle unstructured data..."
summary = summarize_technical_docs(raw_text)
print(summary)

2. Structured Data Extraction using Prompt Engineering

One of the most powerful applications of Gen AI in Data Engineering is turning unstructured text into structured formats like JSON. This allows for seamless integration into downstream databases and analytics platforms.

Python / JSON 
import json

def extract_entities(text):
    model = genai.GenerativeModel('gemini-pro')
    
    # Clearly define the expected structure in the prompt
    prompt = """
    Extract the following entities from the text in JSON format:
    - Company Name
    - Technology Stack (list)
    - Industry Focus
    
    Text: {text}
    """
    
    response = model.generate_content(prompt.format(text=text))
    
    # Parse the string response into a Python dictionary
    try:
        data = json.loads(response.text)
        return data
    except Exception as e:
        print("Error parsing JSON:", e)
        return None
